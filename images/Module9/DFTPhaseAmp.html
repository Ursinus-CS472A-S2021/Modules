import numpy as np
import matplotlib.pyplot as plt

We saw that it's possible to detect a particular sinusoid if we check both the sine and the cosine at its frequency. Now we're going to use this observation to devise a mathematical operator known as the Discrete Fourier Transform (DFT) which checks over all possible pure sines and cosines that go through an integer number of samples over a particular interval. As it turns out, it's sufficient to check ceil(N/2) such cosines and ceil(N/2) such sines, for N samples. Below is an example where there are $N = 40$ samples, and we have each of 20 sines and cosines

Real DFT Definition¶

Now let's define the DFT Given a signal $x$ with $N$ samples

  • Cosine parts: Dot product of $x$ with different cosines

$c[k] = \frac{2}{N} \sum_{n=0}^{N-1} x[n] \cos(2 \pi k \frac{n}{N}), k = 0, 1, ... ceil(N/2)$¶

  • Sine parts: Dot product of $x$ with different sines

$s[k] = \frac{2}{N} \sum_{n=0}^{N-1} x[n] \sin(2 \pi k \frac{n}{N}), k = 0, 1, ... ceil(N/2)$¶

Let's translate this into code now. We simply need to loop over ceil(N/2) sines and cosines, and we store the dot products with each in arrays

def get_dft(x):
    """
    Parameters
    ----------
    x: ndarray(N)
        Signal of interest
    
    Returns
    -------
    c: ndarray(ceil(N/2)), s: ndarray(ceil(N/2))
        Cosine and sine components of the DFT
    """
    N = len(x)
    n_freqs = int(np.ceil(N/2))
    n = np.arange(N)
    c = np.zeros(n_freqs)
    s = np.zeros(n_freqs)
    for k in range(n_freqs):
        c[k] = np.sum(x*np.cos(2*np.pi*k*n/N)) / (N/2)
        s[k] = np.sum(x*np.sin(2*np.pi*k*n/N)) / (N/2)
    return c, s

Amplitude And Phase¶

Recall that a general sinusoid of amplitude $A$ and phase $\phi$ can be broken down as follows

$y[n] = A \cos(2 \pi f t + \phi) = A \cos(\phi) \cos(2 \pi f t) - A \sin(\phi) \sin(2 \pi f t) $¶

$y[n] = a \cos(2 \pi f t) + b \sin(2 \pi f t)$¶

For shorthand, define $a$ and $b$ to be as follows

$a = A \cos(\phi), b = A \sin(\phi)$¶

Amplitude¶

We can recover amplitude as follows:

$a^2 + b^2 = A^2\cos^2(\phi) + A^2\sin^2(\phi) = A^2(\cos^2(\phi) + \sin^2(\phi))$

$a^2 + b^2 = A^2$

Therefore,

$A = \sqrt{a^2 + b^2}$

Phase¶

We can recover phase as follows

$\phi = \tan^{-1} \frac{-b}{a} = \tan^{-1} \frac{-A \sin(\phi)}{A \cos(\phi)} = \tan^{-1} \frac{-\sin(\phi)}{\cos(\phi)} $¶
