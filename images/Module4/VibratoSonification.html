Vibrato And Sonification¶

Vibrato is a periodic wobbling around a central frequency, which is often used for artistic purposes in singing and different musical instruments. We can represent it as a constant instantaneous frequency plus a periodic offset from that frequency. We can then take the antiderivative of this and hear the sound

sr = 44100
t = np.arange(sr*3)/sr
f = 440*t + (10/(12*np.pi))*np.sin(12*np.pi*t)
y = np.cos(2*np.pi*f)
ipd.Audio(y, rate=sr)

Ex) Let's say we wanted the wobbling to get more intense over time. Well, we could add on a chirp to the base frequency to accomplish this. Then when integrating (numerically using cumsum), we will get a chirp in a chirp

$f'(t) = 440t + \cos(12 \pi t^2)$

df = 440 + 10*np.cos(12*np.pi*t**2)
f2 = (t[1]-t[0])*np.cumsum(df)
y2 = np.cos(2*np.pi*f2)
ipd.Audio(y2, rate=sr)
